## 第一章 计算机系统漫游

这一章通过研究"hello, world"这个简单程序的生命周期, 介绍计算机系统的主要概念和主题

### 信息就是位(bit)+上下文

系统中的所有信息都是使用一串 **比特** 表示的, 而区分不同数据对象的唯一方法是根据我们读到这些数据对象时的上下文来判断

### 编译系统

以如下所示的 `hello.c` 程序为例, 观察其被翻译成可执行文件 `hello` 的过程
```c
#include <stdio.h>
int main()
{
printf("hello, world\n");
return O;
}
```

整个翻译过程可以被分为四个阶段
* 预处理阶段
* 编译阶段
* 汇编阶段
* 链接阶段
